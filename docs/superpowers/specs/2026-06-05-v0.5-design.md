## 🎯 v0.5 的定位

v0.1 跑通了"我是什么人格"，v0.2 加了"我和上周比变了什么"。

**v0.5 = 站在巨人肩膀上，做架构现代化**：
- 用 [browser-history](https://github.com/browser-history/browser-history) 替代自实现的 Chrome/Edge/Firefox SQLite 适配器（节省 200+ 行）
- 把 `scripts/` 中间层提到顶层 `digital_nutrition/` package（解决 11 个 `sys.path.insert` 散弹）
- 加 PNG 分享卡（浏览器端 canvas，零新依赖）
- 加 JSON 导出（为未来 sync / 备份打基础）

**v0.5 不做的事**（YAGNI）：
- 规则引擎（v0.1/v0.2 的 dict 合并够用）
- 多维度人格（v0.2 的 7 种够用）
- AFK 检测（需 OS 钩子，超范围）
- Pillow PNG（用浏览器按钮代替）
- Server / REST API（我们不是 daemon）
- Firefox（browser-history 已支持，**白送**）

### 1.3 Source 协议

```python
# digital_nutrition/sources/base.py
from typing import Protocol
from datetime import datetime
from ..models import Event


class Source(Protocol):
    """数据源协议 - 所有数据源实现统一接口"""
    name: str

    def is_available(self) -> bool: ...
    def collect(self, since: datetime | None = None) -> list[Event]: ...


# digital_nutrition/sources/__init__.py
from .browser import BrowserSource
from .git import GitSource

DEFAULT_SOURCES = [BrowserSource(), GitSource()]
```

```python
# digital_nutrition/sources/browser.py
from datetime import datetime
from typing import Optional
import browser_history
from .base import Source
from ..models import Event


class BrowserSource(Source):
    name = "browser"

    def is_available(self) -> bool:
        return True  # browser-history 内部扫描路径

    def collect(self, since: Optional[datetime] = None) -> list[Event]:
        from browser_history import get_history
        outputs = get_history()
        events = []
        for dt, url in outputs.histories:
            if since and dt < since:
                continue
            events.append(Event(
                timestamp=dt,
                duration_seconds=0,    # browser-history 不提供
                source="browser",
                category="other",       # 待分类
                subcategory=None,
                title="",              # browser-history 不提供
                url_or_path=url,
            ))
        return events
```

> ⚠️ browser-history 的默认 API 不返回 title 和 duration。v0.5 接受"无 title / duration=0"降级。

## 📦 Phase 2：分享卡 + 导出（2h）

### 2.1 PNG 分享卡（1h）

**方案**：浏览器端 canvas 绘制 → `toDataURL('image/png')` → 触发下载
**优势**：零新依赖（Pillow / Playwright 都不需要）

`digital_nutrition/report/share.py`:
```python
# 占位：前端逻辑在 report.html.j2 里，Python 端只提供 metadata
def get_share_card_metadata(report_data: dict, persona: str) -> dict:
    """给前端 canvas 绘制提供结构化数据"""
    return {
        "persona": persona,
        "primary_category": max(report_data["by_category"], key=report_data["by_category"].get),
        "pct_by_category": {
            cat: round(seconds / max(report_data["total_seconds"], 1) * 100, 1)
            for cat, seconds in report_data["by_category"].items()
        },
        "top_insights": report_data.get("insights", [])[:3],
        "generated_at": report_data.get("generated_at", ""),
    }
```

`templates/report.html.j2` 末尾加：

### 2.2 JSON 导出（1h）

`digital_nutrition/history/export.py`:
```python
import json
from datetime import datetime
from pathlib import Path
from .store import load_history


def export_all_reports(output_path: Path, limit: int = 1000) -> Path:
    """导出所有历史报告为单个 JSON 文件"""
    reports = load_history(limit=limit)
    payload = {
        "exported_at": datetime.now().isoformat(),
        "version": "0.5.0",
        "report_count": len(reports),
        "reports": reports,
    }
    output_path.write_text(
        json.dumps(payload, ensure_ascii=False, indent=2),
        encoding="utf-8",
    )
    return output_path
```

`digital_nutrition/cli.py` 加 subcommand:
```python
def cmd_export(args):
    from .history.export import export_all_reports
    output = Path(args.output or "reports-backup.json")
    path = export_all_reports(output)
    print(f"✅ Exported {path.stat().st_size} bytes to {path}")
```
